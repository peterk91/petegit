List View
All the Lists can be viewed By their date submitted, module id or number of registered students by clicking on the relative button.
Pending
This shows a list of all the rooms that have an outstanding request. These rooms are not definite and are liable to change. You may chose to delete these requests by clicking 'delete' and they will no longer be sent through to the administrator. You may also view more information, this will tell you the module code, name, number of students, requested rooms and facilities as well as providing you with a timetable displaying this information.
Allocated
These rooms have been booked and accepted by the administrator. If you wish to change these requests you will have to get administrators permission using the release request button.
Denied
These rooms have been rejected and you will be required to re-submit these requests.

Timetable View
This shows a timetable for each module. You can select the module from the list called module code, then chose the week you wish to view. If the Blocks are in green then the request has been accepted, orange shows a pending request and red shows denied requests.
